# Forms Demo – Django Web Framework

## Overview
Forms Demo is a simple Django application that demonstrates how to create and process web forms using Django’s form system. The project shows how user input can be collected, validated, and displayed using Django’s built-in form handling features.

This project is designed as a learning example to understand Django forms, request handling, and the interaction between views, templates, and URL routing.

## Features
- Create and render forms using Django
- Handle GET and POST requests
- Perform form validation
- Display submitted form data
- Demonstrates Django views, templates, and URL routing

## Tech Stack
- Language: Python
- Framework: Django
- Frontend: HTML, CSS
- Database: SQLite (default Django database)
